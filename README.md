# NextUp – Digital Queue Management System

NextUp is a **web-based Digital Queue Management System** built using the **MERN stack (MongoDB, Express.js, React, Node.js)**.  
It helps reduce physical waiting lines by allowing users to generate **virtual tokens** and track queue status in real time, while administrators can efficiently manage and call the next token.

---

## Features

- User **Signup and Login**
- **Digital Token Generation**
- View **Current Token Being Served**
- See **Number of People Ahead in Queue**
- **Admin Dashboard** to call next token
- **FIFO Queue Management**
- **Real-time updates** using Socket.IO
- Clean and responsive user interface
